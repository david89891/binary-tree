# binary-tree
二元樹常用函式以及git指令練習
常用指令

1. git init

2. git add "." or "檔案"
work dir -> staging area
. 代表全部

3. git commit [-m] "留言"
staging area -> local repo

3-1. git log
git commit 指令所更動的歷史紀錄

4. git checkout
local repo -> work dir

5. git merge
local repo -> work dir


git push "地址" or ""
local repo -> remote repo
git pull "地址"
remote repo -> local repo
